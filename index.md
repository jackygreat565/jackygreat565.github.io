## 🛠 Troubleshooting Common Issues

If the software fails to start, check the following items. Most issues originate from missing dependencies or incorrect path settings.

*   **Missing Libraries:** If you see an error about a specific module, ensure you ran the `pip install` command inside the folder where the `requirements.txt` file sits.
*   **Version Mismatch:** Ensure you use Python 3.10. Older versions of Python often lack the necessary structures to support the machine learning libraries present in these projects.
*   **Path Errors:** Command Prompt requires the exact path to your project folder. Use the "Copy address as text" function in Windows File Explorer if you struggle to locate the directory.
*   **Antivirus Interference:** Sometimes, security software flags custom scripts. If the program closes unexpectedly, check your antivirus logs to see if it blocked the execution of the Python interpreter.
